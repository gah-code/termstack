# TermStack Brand Guidelines

## Brand Position

TermStack is a lightweight learning glossary for technical, CMS, content strategy, SEO, frontend, and AI workflow terms.

It should feel calm, intelligent, structured, and useful.

## Brand Mood

TermStack should feel like:

- A quiet study desk
- A technical field guide
- A content systems notebook
- A practical learning companion

## Visual Keywords

- Warm neutral
- Ink black
- Soft parchment
- Muted brass
- Editorial spacing
- Calm interface
- Clear hierarchy

## Color Direction

### Light Mode

The light theme should use warm parchment backgrounds, soft surfaces, ink-like text, and muted brass accents.

### Dark Mode

The dark theme should use deep ink backgrounds, warm text, muted borders, and brass highlights.

## Typography Direction

TermStack should prioritize reading clarity.

Headlines should feel confident and editorial. Body text should feel calm and spacious. Definitions should be easy to scan, while blog posts should feel comfortable for deeper study.

## Interaction Direction

Interactions should feel subtle and quiet.

Use:

- Smooth reveal animations
- Clear focus states
- Soft hover movement
- Reduced-motion support
- Large enough touch targets

Avoid:

- Loud motion
- Harsh contrast
- Dashboard clutter
- Overly decorative UI

## Content Voice

The voice should be:

- Clear
- Practical
- Brief when defining
- Expansive when teaching
- Technical without being cold
- Strategic without being vague

## Example Voice

Instead of:

> A content model is a schema for structured entities.

Prefer:

> A content model defines the reusable structure of content so teams can publish, reuse, and govern it more clearly.
